# Smart Q&A Bot

### Project Info

| **Term:**         | Fall 2025          |
| ----------------- | ------------------ |
| **Project Lead:** | Bryan Kuang        |
| **Team Members:** | Andrew Wu, kodxity |

### **Project Description**

The Smart Q&A Bot acts as an intelligent interface for the team's internal documentation hosted on Outline. By leveraging AI, the bot can understand natural language questions, determine which collection contains the relevant information, and retrieve specific document content to answer the user within Discord.

### **Key Features**

- **Outline Integration:** Direct access to collections and bottom-level document hierarchies.
- **AI Routing:** Analyzes questions to identify the most helpful documentation collection.
- **Path Parsing:** Automatically parses document names in "Parent - Sub - Doc" formats to locate specific files.
- **Knowledge Retrieval:** Opens and reads Outline documents to provide contextualized AI responses.
