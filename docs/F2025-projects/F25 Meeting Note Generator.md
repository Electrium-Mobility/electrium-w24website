# Meeting Note Generator

### Project Info

| **Term:**         | Fall 2025         |
| ----------------- | ----------------- |
| **Project Lead:** | Bryan Kuang       |
| **Team Members:** | Tanish S., Vishal |

### **Project Description**

This automation tool streamlines the documentation of team meetings. It captures voice data, converts it to text using industry-leading transcription services, and automatically compiles the results into a centralized knowledge base without manual intervention.

### **Key Features**

- **Deepgram Transcription:** Integrated high-accuracy voice-to-text service.
- **Google Docs Integration:** Automatically appends meeting summaries to specific Google Docs while preventing data overwriting.
- **Configurable Environment:** Replaced hardcoded paths with configurable lookups for Opus DLLs, making the bot easier to deploy across different systems.
