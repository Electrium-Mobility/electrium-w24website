### **Project Description**

Conversion Kit for converting standard bicycles into E-bikes by installing a central motor near the pedals which provides higher climbing ability and higher torque alongside a battery, speed control system, and a user display.

### **Key Features**

- Central motor near the pedals (mid-drive configuration)
- Higher climbing ability
- Higher torque
- Battery system
- Speed control system
- User display
